## Machine Learning-Based IoT Wearable Device for Epileptic Seizure Detection
An IoT wearable device based on machine learning, which detects epileptic seizures through various sensors as EDA and Accelerometer, then alerts the patients and their relative through a smartphone application to take the necessary precautions.
